### 模型计费

点击某一个供应商，进入模型管理，点开要设置模型旁边的编辑图标，进去设置计费即可。

[知识库](https://www.buildingai.cc/docs/introduction/summary#%E7%9F%A5%E8%AF%86%E5%BA%93)
--------------------------------------------------------------------------------------

用于管理和创建知识库，可以导入不同格式的文件资料，使用AI进行自动清洗和向量化处理后，结合智能体使用，从而增强AI对于回答的精准度。使用场景包括企业智能客服、产品知识问答、企业文档查询等等，支持对知识库进行多角色权限控制，有效进行数据隔离。

### [创建知识库](https://www.buildingai.cc/docs/introduction/summary#%E5%88%9B%E5%BB%BA%E7%9F%A5%E8%AF%86%E5%BA%93)

进入工作台，找到知识库菜单，点击「创建知识库」

![数据看板](https://www.buildingai.cc/summary/image%20(23).png)

输入知识库名称、描述、选择Embedding（向量）模型、然后上传文件即可。

![数据看板](https://www.buildingai.cc/summary/image%20(24).png)

如何设置Embedding（向量）模型？

要开启模型类型为Embedding模型的，才可以在知识库的Embedding模型中选择

![数据看板](https://www.buildingai.cc/summary/image%20(25).png)![数据看板](https://www.buildingai.cc/summary/image%20(26).png)

### [知识库权限](https://www.buildingai.cc/docs/introduction/summary#%E7%9F%A5%E8%AF%86%E5%BA%93%E6%9D%83%E9%99%90)

个人创建的知识库，可以邀请其他成员（有后台管理员身份的用户）一起协作管理；有权限的成员才能看到你创建的知识库：

*   管理者：拥有全部权限，管理整个知识库的全部数据和信息
    
*   编辑者：只能操作文档功能，增删改查自己的文档数据，不能修改他人的
    
*   查看者：只能查看知识库所有数据，不能编辑
    

![数据看板](https://www.buildingai.cc/summary/image%20(27).png)

[MCP工具](https://www.buildingai.cc/docs/introduction/summary#mcp%E5%B7%A5%E5%85%B7)
----------------------------------------------------------------------------------

用于管理和创建 MCP 服务，支持手动创建和使用 JSON 导入，支持 SSE 类型的链接模式。MCP 服务会显示在前台「对话」功能的「工具」中，用户可选择合适的 MCP 工具来增强AI回答效果，例如使用"联网搜索"MCP工具，AI回答会自动联网获取最新数据来回复用户，有效增强回答的实时性。

### [前台创建MCP](https://www.buildingai.cc/docs/introduction/summary#%E5%89%8D%E5%8F%B0%E5%88%9B%E5%BB%BAmcp)

在前台个人中心找到「MCP设置」，点击快速创建 或 JSON导入其中一种方式，按照提示填写好信息即可

![数据看板](https://www.buildingai.cc/summary/image%20(28).png)

### [订阅计划](https://www.buildingai.cc/docs/introduction/summary#%E8%AE%A2%E9%98%85%E8%AE%A1%E5%88%92)

会员等级说明

*   启用会员功能，用户在前台才能访问该功能
    
*   同一时间购买相同等级的会员，只能拥有一个生效中的订阅，订阅时间会叠加
    
*   同一时间购买不同等级的会员，等级高的的订阅将覆盖原有订阅的剩余有效期和权益。 默认显示等级最高的会员
    

![数据看板](https://www.buildingai.cc/summary/image%20(70).png)

平台提供灵活的订阅周期供您选择：

*   月度（30天有效期）
*   季度（90天有效期）
*   年度（365天有效期）
*   永久（永久有效期）
*   自定义时长（自定义天数有效期）
