<Update label="2025-12-15" description="v25.1.0">
  ## 25.1.0 版本发布

  <Warning>
    此次更新会停用所有应用插件，需在应用管理更新插件后重新启用。
  </Warning>

  ### 更新方式

  **Docker部署**
  <Steps>
    1. git拉取最新代码：
       ```bash
       git reset --hard HEAD && git pull
       ```
    2. 重启Docker中buildingai下的Nodejs容器：
       ```bash
       docker restart buildingai-nodejs
       ```
  </Steps>

  **手动部署**
  <Steps>
    1. 本地有pnpm和git则先执行 `pnpm sync-env`，然后执行：
       ```bash
       pnpm bd:update-git && pnpm restart
       ```
    2. 或者执行：
       ```bash
       git reset --hard HEAD && git pull && pnpm predeploy && pnpm restart
       ```
  </Steps>

  ### 🚀 新增
